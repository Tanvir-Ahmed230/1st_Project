# In IUT, CGPA Matters
A modular C project for calculating CGPA.

##Linux
Build:
```
chmod+xmake.sh
./make.sh
```
Test:
```
chmod+xtest.sh
./test.sh
```
Run:
```
chmod+xcalculator.out
./calculator.out
```
##Windows(PowerShell)
Build:
```
.\make.ps1
```
Test:
```
.\test.ps1
```
Run:
```
.\calculator.exe
```
